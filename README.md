# MyVimStyle

## Installation

1. Clone `.vim` to your home directory.

    ```bash
    git clone git@github.com:dq042000/MyVimStyle.git
    ```

2. Make a MyVimStyle link to vimrc.

    ```bash
    ln -s ~/.vim/.vimrc ~/.vimrc
    ```

3. Install Plugins

    ```bash
    vim +PlugInstall +qall
    ```
